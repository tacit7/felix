    RouteWiseApi.Caching.Config (phoenix_backend v0.1.0)

    View Source

    Environment-aware cache configuration management.

    Handles backend selection, TTL policies, and supervision strategies based on Mix environment and application configuration.

    Summary

    Functions

    backend()

    Get the cache backend module for the current environment. Returns the configured backend or falls back to environment default.

    cache_warming_config()

    Get environment-specific cache warming configuration.

    debug_enabled?()

    Check if debug logging is enabled for cache operations.

    invalidation_config()

    Get cache invalidation strategy configuration.

    metrics_enabled?()

    Check if metrics collection is enabled.

    redis_config()

    Get Redis configuration with proper OTP supervision settings.

    supervision_strategy()

    Get supervision strategy for cache backends. Development: :one_for_one (faster restart) Production: :rest_for_one (coordinated restart)

    ttl(category)

    Get TTL for a specific cache category with environment multiplier applied. Categories: :short, :medium, :long, :daily

    ttl_seconds(category)

    Get TTL in seconds for external cache systems (Redis).
