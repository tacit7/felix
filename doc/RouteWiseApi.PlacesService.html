    Functions

    autocomplete_places(input, opts \\ [])

    Get autocomplete suggestions for place search.

    This is always fetched fresh from Google Places API as suggestions are highly contextual.

    Examples

    iex> autocomplete_places("San Franc")
{:ok, [%{description: "San Francisco, CA, USA", place_id: "..."}, ...]}

    cleanup_old_cache(hours \\ 48)

    Clean up old cached place data.

    Examples

    iex> cleanup_old_cache()
:ok

    find_places_by_type(location, place_type, opts \\ [])

    Find places by type near a location.

    Examples

    iex> find_places_by_type(%{lat: 37.7749, lng: -122.4194}, "restaurant")
{:ok, [%Place{}, ...]}

    get_photo_url(photo_reference, opts \\ [])

    get_place_details(google_place_id, opts \\ [])

    Get detailed place information by Google Place ID.

    Checks cache first, then fetches from Google Places API if needed or cache is stale.

    Examples

    iex> get_place_details("ChIJN1t_tDeuEmsRUsoyG83frY4")
{:ok, %Place{}}

    search_places(query, location, opts \\ [])

    Search for places with intelligent caching.

    First checks local cache, then falls back to Google Places API if needed. Caches results locally for faster subsequent requests.

    Examples

    iex> search_places("restaurants", %{lat: 37.7749, lng: -122.4194})
{:ok, [%Place{}, ...]}
